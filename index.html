Uso del sitio web

El uso de este sitio web se rige por las siguientes pautas:

  1. No utilizar el sitio web para fines ilegales o no autorizados.
  2. No interferir con la seguridad del sitio web ni acceder a cuentas de otros usuarios.
  3. Mantener la confidencialidad de tus credenciales de inicio de sesión.
  4. No compartir tus credenciales de inicio de sesión con terceros.
  5. Utilizar contraseñas seguras y actualizarlas regularmente.
  6. No hacer uso indebido de la información personal de otros usuarios.
  7. No realizar actividades de hacking, phishing o intentar acceder a datos sensibles.
  8. Notificar cualquier actividad sospechosa o violación de seguridad.
